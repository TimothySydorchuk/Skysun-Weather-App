# Skysun — Weather App
A clean, responsive weather application built with HTML, CSS, JS and the OpenWeatherMap API.

---

## Features
- Search any city in the world
- Current temperature, feels like, humidity, wind, pressure
- Hourly forecast for the few hours
- 7-day forecast
- Sunrise & sunset times with visual progress bar
- UV index, cloud cover, visibility
- Dynamic background that changes based on temperature
